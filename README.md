# TwinkleParade2D

[English](#english) | [日本語](#日本語)

## English

A 2D claw machine game demo developed with Cocos Creator and TypeScript.

This project is currently under development. It will be continuously improved with additional gameplay features, visual effects, UI refinements, and support for multiple screen layouts.

### Live Demo

🎮 [Play TwinkleParade2D](https://wxy-0104.github.io/TwinkleParade2D/)

### Features

- 2D claw machine gameplay
- Prize-catching mechanics
- Interactive UI and animations
- Physics-based object interactions
- Web browser support
- Desktop and mobile layout support planned

### Technology

- Cocos Creator 3.8.6
- TypeScript
- Web Desktop
